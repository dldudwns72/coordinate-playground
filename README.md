좌표 계산기 요구사항 (선 길이)

- [ ] 좌표 입력 정보
  - [ ] X,Y 좌표 모두 0 ~ 24까지 입력 가능하다.
  - [ ] 좌표 정보는 괄호( , )로 둘러쌓여 있다. EX (10,10)
  - [ ] 쉼표(,)로 x값과 y값을 구분한다.
  - [ ] 입력 범위 초과 시 에러 문구 출력 후 다시 입력한다.
    
- [ ] 좌표 표시
  - [ ] 입력한 좌표에 특수문자를 표시한다.
  - [ ] 좌표값과 좌표값 사이는 '-' 문자로 구분한다.
    

- [ ] 거리 표기
  - [ ] 두 점을 있는 직선의 거리를 표기한다.
